## Physics Options

The `physicsOptions` prop allows you to customize the physics behavior:

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `restitution` | `number` | `0.8` | Bounciness of words (0-1) |
| `frictionAir` | `number` | `0.01` | Air resistance |
| `friction` | `number` | `0.2` | Surface friction |
| `density` | `number` | `0.001` | Mass density of words |

## Initial Velocity

Control the initial velocity when words start falling:

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `x` | `number` | `5` | Horizontal velocity range |
| `y` | `number` | `0` | Vertical velocity range |
| `angular` | `number` | `0.05` | Angular velocity range |
