## Usage

### Gradient Border Card

```tsx
import { GradientBorderCard } from "@/components/ui/gradient-border";

function MyComponent() {
  return (
    <GradientBorderCard
      colors={["#f43f5e", "#ec4899", "#8b5cf6", "#f43f5e"]}
      duration={4}
      glowEffect
    >
      <div className="p-6">
        <h3 className="font-semibold">Card Title</h3>
        <p className="text-sm text-muted-foreground">Card content</p>
      </div>
    </GradientBorderCard>
  );
}
```

### Gradient Border Button

```tsx
import { GradientBorderButton } from "@/components/ui/gradient-border";

function MyComponent() {
  return (
    <GradientBorderButton
      colors={["#22c55e", "#10b981", "#14b8a6", "#22c55e"]}
      duration={3}
    >
      Click me
    </GradientBorderButton>
  );
}
```

### Shimmer Border

```tsx
import { ShimmerBorder } from "@/components/ui/gradient-border";

function MyComponent() {
  return (
    <ShimmerBorder color="hsl(var(--primary))" duration={2}>
      <div className="p-6">
        <h3 className="font-semibold">Shimmer Effect</h3>
        <p className="text-sm text-muted-foreground">Content with shimmer border</p>
      </div>
    </ShimmerBorder>
  );
}
```

### Pulse Border

```tsx
import { PulseBorder } from "@/components/ui/gradient-border";

function MyComponent() {
  return (
    <PulseBorder color="hsl(var(--primary))" duration={2}>
      <div className="rounded-xl border bg-card p-6">
        <h3 className="font-semibold">Pulse Effect</h3>
        <p className="text-sm text-muted-foreground">Content with pulsing border</p>
      </div>
    </PulseBorder>
  );
}
```

## Notes

- All gradient border components support custom color arrays for the gradient effect
- Animation duration can be customized for different speeds
- Border width and radius are configurable for various styles
- Glow effects add an outer shadow for enhanced visual impact
- Shimmer borders create a single rotating highlight effect
- Pulse borders create a breathing/glowing animation
- All components are fully customizable with additional CSS classes
