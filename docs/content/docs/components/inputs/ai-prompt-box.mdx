## Features

The AI Prompt Box component includes several features:

- **Text Input**: Auto-resizing textarea with keyboard shortcuts (Enter to send, Shift+Enter for new line)
- **File Upload**: Drag & drop or click to upload images with preview
- **Voice Recording**: Built-in voice recording with visualizer
- **Search Mode**: Toggle to search the web
- **Think Mode**: Toggle for deep thinking prompts
- **Canvas Mode**: Toggle for creative canvas interactions
- **Clipboard Paste**: Paste images directly from clipboard
