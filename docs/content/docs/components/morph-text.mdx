## Features

- **Multiple Effects**: Includes Blur Morph, Character Stagger, Scramble, and Velocity Morph.
- **Customizable**: Adjust durations, delays, intervals, and character sets.
- **Accessible**: Uses standard HTML elements and respects reduced motion preferences (via Framer Motion).
- **Responsive**: Works with any text size and font.
