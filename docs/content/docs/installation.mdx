---
title: Installation
description: Get started with Joly UI in under 2 minutes. Add 50+ React components to your Next.js or React project with a single command.
---

<Callout>
  **Prerequisites:** You need a project with [shadcn/ui](https://ui.shadcn.com/docs/installation) already configured.
</Callout>

<Steps>
<Step>

### Initialize shadcn/ui (if not already done)

Run the `init` command to set up shadcn/ui in your project:

```package-install
npx shadcn@latest init
```

</Step>
<Step>

### Add Joly UI Components

Use the shadcn CLI to add any Joly UI component directly from our registry:

```package-install
npx shadcn@latest add "https://jolyui.dev/r/vercel-tabs"
```

The component will be added to your `components/ui` folder, ready to customize.

</Step>
<Step>

### Use the Component

Import and use the component anywhere in your project:

```tsx
import { Button } from '@/components/ui/button';

export default function Home() {
  return (
    <div>
      <Button>Click me</Button>
    </div>
  );
}
```

</Step>
</Steps>
