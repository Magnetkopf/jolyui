## Why Joly UI?

Joly UI extends shadcn/ui with **50+ additional components** that you won't find in the original library. Same copy-paste experience, same quality, more options.

- **Zero dependencies to install** - Just copy the code into your project
- **Built on battle-tested primitives** - Uses Radix UI and shadcn/ui foundations
- **TypeScript-first** - Full type safety with strict mode support
- **Tailwind CSS v4 ready** - Modern styling with CSS variables
- **WCAG 2.1 AA compliant** - Accessible by default

## Key Features

- **Composable by design** - Build complex interfaces by combining simple, focused components
- **Extends shadcn/ui** - Same installation process, same conventions, more components
- **Copy-paste ready** - No npm packages, just copy the code you need
- **Fully customizable** - Built with Tailwind CSS and designed to match your design system
- **Advanced components** - Data tables, media players, drag-and-drop, text animations, and more

## Get Started in 2 Minutes

```bash
npx shadcn@latest add "https://jolyui.dev/r/button"
```

That's it. The component is now in your project. Customize it however you want.
